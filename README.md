# raga-csl
Citation Style Language (CSL) file for the *Revista de la Asociación Geológica Argentina (RAGA)*.

This repository contains a Citation Style Language (CSL) file implementing the bibliographic and formatting guidelines of the Revista de la Asociación Geológica Argentina (RAGA).
It can be used with Zotero, Mendeley, Pandoc, Quarto, and other CSL-compatible tools.
The style aims to faithfully reproduce the journal’s official author-date citation and reference format.

*Archivo* Citation Style Language (CSL) *para la Revista de la Asociación Geológica Argentina (RAGA).*

*Este repositorio contiene un archivo* Citation Style Language (CSL) *que implementa las normas bibliográficas y de formato de la Revista de la Asociación Geológica Argentina (RAGA).
Puede utilizarse con Zotero, Mendeley, Pandoc, Quarto y otras herramientas compatibles con CSL.
El estilo busca reproducir fielmente el formato oficial de citación en el texto (autor-año) y de referencias bibliográficas requerido por la revista.*
